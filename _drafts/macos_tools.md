## iTerm2

### 快捷键

```console
Ctrl + A - 光标移到行首

Ctrl + E - 光标移到行尾

Ctrl + W - 清除光标之前一个单词

Ctrl + K - 清除光标到行尾的字符

Ctrl + H - 删除光标前一个字符

Ctrl + F - 光标后移一个字符 (右箭头

Ctrl + B - 光标前移一个字符 (左箭头
```

```console
Option + b - 前移至上一个空格 (前移至上一个单词开头

Option + f - 后移至下一个空格 (后移至下一个单词开头
```
