## 选择器

### 通配符选择器

```css
* {
  color: red;
}
```

### 元素选择器

```css
div {
  color: red;
}
```

### 类选择器

```css
.class-name {
  color: red;
}
```

### ID 选择器

```css
#id-name {
  color: red;
}
```

### 属性选择器

```css
[disabled="true"] {
  color: red;
}
```

### 伪类选择器

```css
:hover {
  color: red;
}
```

### 派生选择器

- 后代元素选择器

  ```css
  p span {
    color: red;
  }
  ```

- 子元素选择器

  ```css
  p > span {
    color: red;
  }
  ```

- 相邻元素选择器

  ```css
  div + p {
    color: red;
  }
  ```
