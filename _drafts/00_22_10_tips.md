## koa2 项目，生成 swagger 在线 API 文档

`koa2-swagger-ui` 可生成 swagger 在线 API 文档

- 读取本地的 API 文档，yaml 或者 json 格式，推荐 yaml 格式；
- 使用 `swagger-jsdoc` 可以将路由的注释解析为 json 格式数据，再生成在线 API 文档。
